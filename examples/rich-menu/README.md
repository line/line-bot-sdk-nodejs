# rich menu

[Using rich menus](https://developers.line.biz/en/docs/messaging-api/using-rich-menus/)

## How to use

### Install deps

``` shell
$ npm run build-sdk
$ npm install
```

### Configuration

``` shell
$ export CHANNEL_SECRET=YOUR_CHANNEL_SECRET
$ export CHANNEL_ACCESS_TOKEN=YOUR_CHANNEL_ACCESS_TOKEN
```

### Run

``` shell
$ node .
```
